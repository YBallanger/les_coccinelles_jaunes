# Les Coccinelles Jaunes

## Project Description

This repository contains the source code for the presentation website of the association "Les Coccinelles Jaunes". The association's goal is to create social connections and enable isolated individuals to meet and share moments together.

This website is a simple front-end application, designed to introduce the association, its missions, activities, and provide necessary information for getting in touch or joining the community.

# Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |
